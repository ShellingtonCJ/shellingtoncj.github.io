Creating a window

Creating a window in Oatmeal is actually quite easy. This is how you would do it.

#include <iostream>
#include "Oatmeal/Oatmeal.hpp"
      
      int main(int argc, char* argv[]) {
      if (Oatmeal::Init() < 0) {
         std::cerr << "Failed to init Oatmeal" << std::endl;
      }
      Window window("Window example", 860, 540);
      window.fill(255, 255, 255);
      while (Oatmeal::running) {
         Oatmeal::update();
         window.clear();
         window.update();
      }
      window.quit();
      Oatmeal::quit();
      return 0;
      }
      
         
So, Let's run through what this does. First of all, we include the main Oatmeal header file
            #include "Oatmeal/Oatmeal.hpp"This provides functionality for basic Oatmeal functions like windows.
            

Next, inside our main function, we try to initialize Oatmeal

if (Oatmeal::Init() < 0)
If the function returns a value less than zero, there was an error. So we check for the error and log it.

std::cerr << "Failed to init Oatmeal" << std::endl;

Next, we create a window. 

Window window("Window example", 860, 540);

The window constructor takes three arguments. A title, a width, and a height. See more about the Window class.


Then, we fill the window with white.

window.fill(255,255,255)

What this does is makes it so whenever the window draws what is on the screen, it will always start by filling the window with white.


Next, while Oatmeal is running, we call its update function to handle Oatmeal's events

while (Oatmeal::running) {
   Oatmeal::update();


Then, we clear the window and render our white to it, with the update function.

window.clear();
window.update();


The reason we clear the window before drawing things on it is that otherwise, we will make the program slow by not deleting things we have already drawn.


If you close the window, Oatmeal::running will not be true and it will exit the loop. Then, we close the window and shtutdown Oatmeal

window.quit();
Oatmeal::quit();
